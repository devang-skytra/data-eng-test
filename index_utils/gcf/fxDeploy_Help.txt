CONCISE
----------

gcloud beta functions deploy 
(NAME : 
--region=REGION) 
[--entry-point=ENTRY_POINT]		 Defaults to the resource name suffix, if not specified
[--memory=MEMORY] 
[--retry] 
[--runtime=RUNTIME] 
[--source=SOURCE]
	Source code in Google Cloud Storage (must be a .zip archive),
	Reference to source repository
		https://source.developers.google.com/projects/${PROJECT}/repos/${REPO} (MASTER BRANCH)
	Local filesystem path (root directory of function source).
[--stage-bucket=STAGE_BUCKET] 	Used when 'Local filesystem path' used for --source
[--timeout=TIMEOUT] 			Defaults to 60 seconds
--set-env-vars=[KEY=VALUE,…]	Can be [KEY=VALUE,…] or --env-vars-file
--trigger-bucket=TRIGGER_BUCKET
